# USAG-Lib

Universal Security Assistant Gear - Library

### macro

프로젝트 보조 코드, 자동화 코드, 설정 값 등을 포함합니다.

### Icons

아이콘 이미지 바이너리를 제공합니다.

### Szip

간단화한 ZIP64 컨테이너 형식을 읽고 씁니다.

### Star

간단화한 TAR-PAX 컨테이너 형식을 읽고 씁니다.

### Bencode

이진 데이터를 문자열로 인코딩하고 다시 디코딩하는 기능을 제공합니다.

### Bencrypt

기본 암호화 기능을 담당합니다.

### Opsec

복합 암호화 기능과 암호파일 형식화를 담당합니다.
